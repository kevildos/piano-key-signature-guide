Piano Key Signature Helper

A web tool created using JavaScript to help familiarize people with key signatures. To learn more about key signatures, click here.

Try recording a melody and see what the possible key signatures are. Select a key signature from the generated list to see it displayed on the piano. The highlighted keys in green is the TONIC, or the root note. The red is the DOMINANT and the yellow keys are the rest of the keys in the scale.

Try playing the highlighted keys focusing on the dominant and tonic notes to produce more of a melodic sound!
